A day, seen at a glance.

Gentle activity colors make the log easier to scan. A shared visual language then turns the same events into a responsive Home overview, a useful Timeline stream, and one playful daily portrait.

Best first version

Pair a desktop ribbon / mobile chapters view on Home with the Activity stream on Timeline. They reuse the same colors and data while staying easy to read.

Ribbon ↔ chapters+Activity stream

One quiet color per activity

Use color as a scan aid, never as the only label. Each event keeps its name and marker; the tint is a gentle, repeatable memory cue. Click a swatch to spotlight it across the mocks.

The palette is deliberately low-chroma. Related activities feel harmonious, but pee and water remain distinguishable at a glance.

Home concept: ribbon to chapters

The 24-hour ribbon works when it has room. Below roughly 640 px, it folds into morning, midday, and afternoon chapters so nearby events stay distinct instead of collapsing into the same pixels.

A chronology that has rhythm

This version makes duration visible without turning the screen into a chart. It also keeps event editing simple: tapping a tinted row can open the same inline editor that exists today.

  • Moments use dotsMeals, pee, water, treats, wake, and notes remain precise points in time.
  • Durations use short barsSleep, walks, training, and play communicate length as well as start time.
  • Tints stay quietText remains dark and readable; saturated color is reserved for the marker.
  • Color is never aloneEvery row keeps a label, symbol, time, and useful secondary detail.

Fun concept: the Daily orbit

A circular 24-hour portrait makes each day feel distinct. It works best as an optional day summary or insight—not as the only way to read the log.

Memorable, but secondary

The orbit turns an ordinary log into a recognizable daily pattern. It can sit at the top of a selected day, appear in Insights, or become a shareable “Zoe’s day” image later.

Build one visual language in three layers.

1 · Foundation

Add stable activity tints and markers to Home recent activity, Timeline rows, and filter choices.

2 · Useful overview

Add the responsive ribbon / chapters overview to Home and as a compact header for each day in Timeline.

3 · Delight later

Prototype the Daily orbit in Insights once enough complete days exist to make it meaningful.

The practical pairing is responsive overview + stream. It improves scanning now, works at 320 px without compressing events, and leaves the orbit as a genuinely fun reveal instead of forcing it into every task.
